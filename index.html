Características

O técnico em informática é um profissional com formação para atuar em diversas frentes da área de tecnologia da informação (TI), seja em empresas especializadas ou seja nos departamentos de TI de empresas dos mais variados portes e ramos de atuação. Se você está pensando em fazer um curso técnico em informática, saiba que está fazendo uma boa escolha, pois a tecnologia é um mercado de trabalho que não para de crescer no Brasil. Agora, se você tem alguma dúvida relativa à grade curricular do curso, acompanhe nos tópicos a seguir e veja o quanto esta formação é dinâmica e interessante!

Duração de 2,5 anos

Aqui você não precisa fazer matérias que não sejam estritamente de formação para você atuar no mercado de trabalho. Curta duração: Um dos grandes diferenciais da proposta do EAD - INTERATIVO é formar profissionais com qualidade e foco. São 36 meses com uma proposta inovadora. Com esses diferenciais nós ensinamos o que o aluno realmente precisa com foco e objetividade durante o curso.

Diferenciais Inovadores

Temos alguns diferenciais que vão transformar a vida dos nossos alunos, que precisam se sentir desafiados constantemente com uma proposta inovadora de ensino.

Você vai aprender

Desenvolver e fazer manutenção de sistemas em Java (JSE, JEE, JME) e Microsoft .NET/C#, utilizando análise orientada a objetos (UML), banco de dados relacionais (Oracle, DB2, PostgreSQL, SQL Server e MySQL. Criar aplicações para Internet utilizando padrões, frameworks e linguagem para desenvolvimento WEB (JSF, JSP, JSTL, HTML, XHTML, XML, JavaScript e jQuery).

Sobre nos

Somos 3 alunos com objetivo de fazer uma mudança em algumas areas no ensino em EAD, este site é uma base ficticia, os reais motivo de realizar este projeto sera explicado em sala dentro da instituição, Guilherme, Mateus e Murilo, somos os responsaveis em realizar a parte teorica e pratica deste projeto...
